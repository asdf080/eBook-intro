# eBook

### 🔎소개
전자책, eBook에 대해 소개하는 홈페이지입니다. 
모든 해상도를 지원하는 반응형 웹으로 제작되었습니다.

### 📅제작 기간
2023.12.14 ~ 2023.12.27

### 🗂개발 환경
- HTML
- CSS
- JavaScript
- jQuery

### 🎈주요 기능
- 다크 모드/라이트 모드 토글 버튼
- 많은 정보를 한눈에 볼 수 있는 캐러셀 슬라이드
- Intersection Observer API를 활용한 스크롤 연동 CSS 애니메이션
